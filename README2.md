The file run\_analysis.R process the files from the *getdata\_projectfiles\_UCI HAR Dataset.zip*.
then it select the Mean and standard deviation variable and generate a summary per subject per activity, named *x\_data\_summmary.txt*.
Check the codebook for further information on the variables.

How to proceed.
unzip the archive and place the run\_analysis.R in the same folder. use this folder as the working directory. Run the script.
